## Errores encontrados y resueltos

- Puerto 3000 ocupado por Grafana → web movida a puerto 3003
- Puerto 5432 conflicto con PG local → Docker mapeado a 5433
- Puerto 8888 ocupado → Docker mapeado a 8889
- `@hono/trpc-server` incompatible con tRPC v11 → reemplazado con fetchRequestHandler
- `FetchCreateContextFnOptions` no tiene `info` → tipo inline custom
- `onSuccess: refetch` type error → wrapping con arrow function
- Stream model no tiene `createdAt` → orderBy cambiado a `streamKey`
